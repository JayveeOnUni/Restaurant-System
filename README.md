## Restaurant Management System (Project-Based Learning)

This project is a **Restaurant Management System** developed in **C# Windows Forms**. It is part of project-based learning at our university, integrating concepts from two subjects:

- **Object-Oriented Programming (OOP)**
- **Data Structures & Algorithms (DSA)**

The system handles basic restaurant operations, focusing on applying OOP principles and efficient data structures to manage functionalities like menu management, order processing, and billing.
